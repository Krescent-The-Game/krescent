Krescent

Drive a moon buggy across an alien surface — dodge rocks, shoot enemies, survive the endless run. Originally Nuxt 2 + BabylonJS 4; playable WebGL port coming in Phase 2.

Clone & play locally Controls & docs

Play locally

Nuxt 2 requires Node 18. From the repo:

  • nvm use 18
  • npm install --legacy-peer-deps
  • npm run dev → open /game

Phase 2 rebirth: BabylonJS 7 + Vite static deploy on Vercel Node 24.
